# Readshift

Set a countdown for your reading session and stay focused until the ring completes. Built for readers who believe serious books deserve uninterrupted time. One screen, one session: start the ring, put the phone face-down, come back when it chimes. No noise, no streaks, no accounts.

iPhone & iPad. Free. No ads, no in-app purchases, no accounts, no network.
